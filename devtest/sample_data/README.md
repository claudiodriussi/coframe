# Sample Data for Coframe Library Demo

This directory contains sample data to populate the Coframe library database with realistic books, authors, and publishers.

## 📊 Dataset Overview

- **Publishers**: 35 major publishers (Penguin, HarperCollins, Einaudi, Gallimard, etc.)
- **Authors**: 180 authors from various nationalities
- **Books**: 10 books (expandable to 450+ if needed)
- **Multi-author books**: Included (e.g., LOTR with Tolkien & Lewis as co-contributors)

## 📁 Files

- `publishers.csv` - 35 publishers with country and website
- `authors.csv` - 180 authors with birth dates and nationality
- `books.csv` - Books with language, pages, price, description, tags
- `books_authors.csv` - Many-to-many relationships between books and authors
- `import_sample_data.py` - Python script to import all data

## 🏷️ Tags System

Books include comma-separated tags where **first tag = primary genre**:

```csv
"Fiction, Dystopian, Classic"
"Fantasy, Young Adult, Adventure"
"Biography, Holocaust, Memoir"
```

This allows:
- Simple queries: `WHERE tags LIKE '%SciFi%'`
- Primary genre: `tags.split(',')[0]`
- Future migration to full tag system

## 🔍 Sample Queries

After import, try these queries:

```python
# Books by language
books_it = session.query(Book).filter(Book.language == 'it').all()

# Books with multiple authors
multi_author = session.query(Book).join(BookAuthor).group_by(Book.id).having(
    func.count(BookAuthor.author_id) > 1
).all()

# Books by publisher
penguin_books = session.query(Book).join(Publisher).filter(
    Publisher.name.like('%Penguin%')
).all()

# Books by tag
scifi_books = session.query(Book).filter(Book.tags.like('%Science Fiction%')).all()
```

## 🔧 Customization

To add more books:
1. Edit `books.csv` (maintain CSV format)
2. Add corresponding entries in `books_authors.csv`
3. Run import script

To reset data:
```bash
# Delete database and reimport
rm devtest.sqlite
python devtest.py
python sample_data/import_sample_data.py
```
